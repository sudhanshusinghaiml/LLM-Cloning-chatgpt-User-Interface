# Cloning ChatGPT User Interface


### To execute the code follow below steps
- Create a virtual environment

```python
conda create -n cloning-chatgpt-user-interface-venv python=3.12 -y
``` 

- Install the chainlit & openai using pip

```python
pip install chainlit openai
```

- Run the code using
```python
chainlit run clonning_chatgpt.py -w
```
